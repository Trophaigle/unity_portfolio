# 🎮 3D Game Experience Portfolio

A modern interactive portfolio built with **Next.js** and **Tailwind CSS**, showcasing immersive projects developed with industry-standard 3D engines such as **Unity** and **Unreal Engine**.

The platform presents my work across **Desktop, Mobile, and AR/VR experiences**, with a focus on interaction design, real-time 3D environments, and user experience in immersive systems.

🌐 Live website: [Visit website](https://unityportfolio-gamma.vercel.app/)

---

## 🚀 Overview

This portfolio is designed to centralize and present my work in:

- 🎮 Real-time 3D games and interactive experiences
- 🖥️ Desktop applications (Unity / Unreal Engine projects)
- 📱 Mobile interactive experiences
- 🥽 AR/VR immersive applications

The goal is to highlight both **technical development** and **user experience design** in interactive 3D environments.

---

## 🛠️ Tech Stack

- **Framework:** Next.js  
- **Styling:** Tailwind CSS  
- **Content Structure:** Component-based architecture  
- **Data Management:** JSON / TypeScript-based content system  
- **3D Engines:** Unity, Unreal Engine (project showcase integration)

---

## 🎯 Key Features

### 🧭 Interactive Project Showcase
Each project is presented with structured information:
- Description
- Technologies used
- Platform (Desktop / Mobile / AR / VR)
- Screenshots / media previews
- Links to builds or demos (if available)

---

### 🎮 Multi-Platform Focus
Projects are categorized by platform:

- 💻 Desktop applications (PC / Mac)
- 📱 Mobile experiences
- 🥽 AR / VR immersive applications

---

### 🧱 Modular Architecture
- Fully component-based Next.js structure
- Reusable UI components for project cards and sections
- Clean separation between data and presentation

---

### 📦 Data-Driven Portfolio
All project content is stored using:
- JSON files
- TypeScript modules

This allows easy updates and scalability without modifying UI logic.

---

## 🧠 Design Philosophy

This portfolio is built around:

- Clarity of presentation
- Focus on interaction and immersion
- Highlighting real-time 3D experience work
- Clean and minimal UI to emphasize content
- Developer-first architecture for scalability
